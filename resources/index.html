Cascade style sheet(Css) table references

Tag Name Description
color: Color The color property is used to set the color of the text
text-indent Text indent The text indent property is used to specify the indentation of the first line of a text
letter-spacing letter spacing The letter spacing property is used to specify the space between the characters in a text.
line-height Line height The line-height property is used to specify the space between lines
word-spacing Word Spacing The word-spacing property is used to specify the space between the words in a text.
white-space White space The white-space property specifies how white-space inside an element is handled.

Text aligment and text direction

Tag Name Description
text-align: Text align The text-align property is used to set the horizontal alignment of a text. A text can be left or right aligned, centered or justified.
text-align-last Text align last The text-align-last property specifies how to align the last line of a text.
direction Text Direction The direction properties can be used to change the text direction of an element
unicode-bidi unicode Helps to handle the text that comes from both directions
vertical-align Vertical aligment The vertical align property sets the vertical aligment of an element
font-size Font Size The font-size property sets the size of the text.
